# Phrases to Remove

## Throat-Clearing Openers

These announce content instead of delivering it. Cut them and start with the actual point.

- "Here's the thing:"
- "The thing is,"
- "The uncomfortable truth is"
- "It turns out"
- "The real [X] is"
- "Let me be clear:"
- "The truth is,"
- "I'll say it again:"
- "I'm going to be honest"
- "Can we talk about"
- "Here's what I find interesting:"
- "Here's the problem:"
- "Here's the thing about [X]:"
- "What I want you to know is"
- "Let's be real:"
- "Let's be honest:"
- "With that said,"
- "That said,"
- "Having said that,"
- "All that to say,"
- "So here's the deal:"

**Fix:** Delete the opener. The sentence that follows is usually the real sentence.

## Emphasis Crutches

These attempt to manufacture weight. They don't add it.

- "Full stop." / "Period."
- "Let that sink in."
- "This matters because"
- "Make no mistake"
- "Here's why that matters"
- "And that's important."
- "That's significant."
- "I can't stress this enough"
- "This is huge."
- "This is a big deal."

**Fix:** If the content is important, the reader will understand that. Emphasis crutches signal the writer isn't trusting the content.

## Hollow Qualifiers

Adjective-adverb pairs that inflate without informing.

- "very unique" (unique is already absolute)
- "absolutely essential" (essential is already absolute)
- "completely different" (different suffices)
- "truly remarkable" (remarkable or it isn't)
- "deeply committed" (committed is a binary)
- "highly innovative" (show the innovation)
- "extremely important" (how important? be specific)
- "incredibly powerful" (quantify the power)

## Business Jargon

Replace with plain language.

| Avoid | Use instead |
|-------|-------------|
| Navigate (challenges) | Handle, address, work through |
| Unpack (analysis) | Explain, examine, break down |
| Lean into | Accept, embrace, do more of |
| Landscape (context) | Situation, field, market |
| Game-changer | Significant shift, major change |
| Double down | Commit harder, invest more |
| Deep dive | Detailed look, analysis |
| Take a step back | Reconsider, zoom out |
| Moving forward | Next, from now on |
| Circle back | Return to, revisit |
| On the same page | Aligned, agreed |
| Bandwidth (personal) | Time, capacity |
| Low-hanging fruit | Easy wins, quick fixes |
| Move the needle | Make a difference, change the numbers |
| Blue-sky thinking | Unconstrained ideas |
| Synergize | Work together |
| Boil the ocean | Do too much |

## Filler Sentence Starters

These open sentences without earning their place.

- "At its core,"
- "In today's [X],"
- "In the world of [X],"
- "In a world where"
- "Interestingly,"
- "Importantly,"
- "Crucially,"
- "When it comes to"
- "The reality is"
- "The fact is"
- "The truth is"
- "It's worth noting that"
- "It's important to understand that"
- "This is a reminder that"
- "Note that"
- "Keep in mind that"

**Fix:** Delete and begin with the actual content.

## Meta-Commentary

Self-referential asides that break the reader's experience.

- "Hint:"
- "Plot twist:"
- "Spoiler:"
- "You already know this, but"
- "But that's a post for another day"
- "[X] is a feature, not a bug"
- "Dressed up as"
- "What I really mean is"

## Telling Instead of Showing

Announcing significance instead of demonstrating it.

- "This is genuinely hard."
- "This is important."
- "This is what [X] actually looks like."
- "This is a big deal."
- "This changes everything."
- "This is what leadership looks like."

**Fix:** Show the thing. If it's hard, describe the difficulty. If it matters, explain why. Don't announce it.

## Summary Openers

Prepending sections with what they'll cover.

- "In this section, we'll look at..."
- "What follows is..."
- "Below, you'll find..."
- "This covers..."

**Fix:** Just say the thing. Readers can see that a section covers what it covers.
