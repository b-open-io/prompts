# Content Strategy Rules

Platform-agnostic rules for social content creation. Extracted from production systems.

## Hooks

- Lead with a concrete observation, shipped thing, or real result. No setup required.
- Specifics beat abstractions: "shipped X, learned Y" beats "here's why Z matters."
- Drop the formula templates. Write how you actually talk.

## X Best Practices

- Max 280 chars per tweet. Be punchy.
- For threads: first tweet must hook standalone. One idea per tweet. 3-5 parts max.
- Avoid external links in tweet body (kills reach on X — put links in replies).
- Specificity > generality. Numbers > adjectives.
- No hashtags (or 1 max). No engagement bait. No corporate speak.
- Suggest an image_prompt when visuals would help.

## What to Avoid

- Do NOT comment on trending things unless you have firsthand experience with them.
- Do NOT use: "this changes everything", "everyone is", "incredible", "amazing", "game-changer."
- Trending context is optional — skip it if you have nothing real to add.
- No manufactured contrarianism. Only take positions you'd actually defend.

## Voice Matching

- Write EXACTLY like the voice examples. Same energy, same style, same vocabulary.
- Voice examples are the real style guide. Match that energy, not the hype.
- Every post should feel like it came from a real person sharing genuine thoughts, not a content calendar.
- Draw from trending topics AND recent work. Connect what's happening in the industry to what you're building.

## Thread Structure

- First post hooks standalone — someone seeing only the first tweet should want to engage.
- One idea per post. Don't pack multiple thoughts into a single tweet.
- Build momentum — each post should make the reader want the next one.
- End with a concrete takeaway or call to action (not engagement bait).
