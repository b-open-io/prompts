# Payment Systems Directory

Reference for all mainstream payment systems. Primary systems (Stripe, BSV) are deeply integrated. Others are listed for awareness and routing.

## Primary Systems (Deep Integration)

### Stripe (Payments, Billing, Connect, Elements)
- Official docs + API: https://docs.stripe.com/
- SDKs: https://docs.stripe.com/sdks
- GitHub org: https://github.com/stripe
- CLI: `brew install stripe/stripe-cli/stripe`

### BSV (Bitcoin SV — direct peer-to-peer payments)
- @bsv/sdk: https://github.com/bitcoin-sv/ts-sdk
- 1Sat Ordinals: https://docs.1satordinals.com/
- Yours Wallet: https://yours.org
- ARC broadcast: https://arc.bitcoinsv.com/

### x402 Protocol (HTTP-native agentic stablecoin payments)
- GitHub: https://github.com/coinbase/x402
- Official standard: https://www.x402.org/
- Coinbase docs: https://docs.cdp.coinbase.com/x402/welcome

### Paddle (merchant-of-record + SaaS billing)

### CREEM (MoR for indie hackers/SaaS — dev-first SDKs)

- Docs + SDKs: https://docs.creem.io/
